# 灯晶の巡礼 仕様書

## ゲーム概要

「灯晶の巡礼」は、ブラウザだけで遊べるオリジナルJRPGです。PixiJS v8で描画し、タイル制の探索、ターン制バトル、装備、店、セーブ/ロード、仲間加入、サイドクエスト、分岐エンディングを1本にまとめています。BGMと効果音はWeb Audioで生成し、外部音源や外部アセットは使っていません。音はデフォルトOFFです。

舞台は、夜を守る灯晶を失った小国。プレイヤーはエルナを操作し、カイ、ミオと合流しながら、雫の洞窟と黒曜塔を巡って星灯りを取り戻します。

## ストーリー

灯丘の里を照らしていた灯晶が、黒曜塔へ奪われました。長老は、塔の闇を開くには雫の洞窟に眠る「青の灯晶」が必要だと語ります。エルナは見張りのカイと旅立ち、洞窟の番人を倒して港への道を開きます。

塩風の港では、灯読みのミオが黒曜塔の鏡の結界を見抜きます。港長からは、青の灯晶を灯台へ分けるか、旅のために温存するかを問われます。この選択は報酬とエンディング文に反映されます。

黒曜塔の最上部で待つのは、属性結界を切り替える「星喰いの鏡」。火、氷、光の弱点を突き、大技を止めながら撃破するとクリアです。

## 操作方法

- 移動: 方向キー / WASD / 画面左下の方向ボタン
- 決定・調べる: Enter / Space / Z / 画面右下のA
- キャンセル・メニュー: Escape / X / 画面右下のB
- メニュー直接表示: M
- メニュー内操作: ボタンをクリック、タップ、またはTabでフォーカスしてEnter

PCはキーボードのみ、スマホはタッチのみで全機能にアクセスできます。

## システム

### マップ

- 灯丘の里: 開始地点。長老、カイ、店、宿、セーブ灯、サイドクエストNPC、宝箱があります。
- 星風の道: 各マップをつなぐフィールド。ランダムエンカウントがあります。
- 雫の洞窟: ダンジョン1。宝箱、古い鍵、押し岩、スイッチ、鉄格子、鍵扉、中ボスがあります。
- 塩風の港: 町2。ミオ加入、港長の分岐選択、店、宿、セーブ灯があります。
- 黒曜塔: ダンジョン2。終盤装備、回復宝箱、ラスボスがあります。

### 探索ギミック

- 宝箱: 所持品や鍵を入手します。一度開けると空になります。
- 古い鍵: 雫の洞窟の奥扉を開けます。
- 押し岩: 岩を押してスイッチに乗せると鉄格子が開きます。
- マップ遷移: 町、フィールド、ダンジョン間を移動します。

### ステータス

- HP: 0になると戦闘不能。
- MP: 特技を使う資源。
- ATK: 物理攻撃力。
- DEF: 被ダメージ軽減。
- MAG: 回復量、魔法攻撃、状態異常成功率に影響。
- SPD: ターン順に影響。

レベルアップでHP/MP/ATK/DEF/MAG/SPDが上がり、キャラクターごとに段階的に特技を覚えます。

### 戦闘

- 最大3人パーティと敵隊列のターン制バトルです。
- 行動順はSPDに乱数を足して決まります。
- コマンドは「攻撃」「特技」「道具」「防御」「にげる」です。
- ボス戦では「にげる」は無効です。
- 会心、属性弱点、属性耐性があります。
- 状態異常は毒と睡眠を実装しています。毒はターン開始時にダメージ、睡眠は行動をスキップしてターン経過で解除されます。
- 全滅時は最後に泊まった宿付近へ戻り、所持金が半減します。

### ラスボスの属性パズル

「星喰いの鏡」は結界を切り替えます。画面と戦闘ログに「結界 / 弱点」が文字で表示されます。

- 結界:氷 / 弱点:火
- 結界:火 / 弱点:氷
- 結界:闇 / 弱点:光

弱点を突くと大技のチャージが散り、結界が切り替わります。

### 店・宿・装備・所持品

- 店では消費アイテム、武器、防具を購入できます。
- 売却は半額。非売品やキーアイテムは売れません。
- 宿はHP/MPと状態異常を回復し、全滅復帰地点を更新します。
- 装備はキャラクターごとの装備可否を持ち、装備変更でステータスに即時反映されます。
- 所持品画面から回復アイテム、毒/睡眠治療アイテムを使えます。

### セーブ/ロード

ブラウザのlocalStorageへ保存します。保存内容は位置、パーティ、レベル、HP/MP、装備、所持品、所持金、フラグ、宝箱、ギミック、設定です。セーブが無い場合や壊れている場合はクラッシュせず、新規開始へ戻れます。

### アクセシビリティ

- キーボードフォーカスを明示表示。
- 弱点や状態異常は色だけでなく文字で表示。
- `prefers-reduced-motion` とゲーム内「演出控えめ」設定を尊重。
- 音はデフォルトOFF。音量とミュート設定を保存。
- 高コントラストのパネル、ボタン、HUDを使用。

## 実装一覧

### 仲間

- エルナ: 主人公。回復と雷術を覚える万能型。
- カイ: 長老イベント後に加入。HP/ATK/DEFが高く、火属性攻撃を覚える。
- ミオ: 港イベント後に加入。MAG/SPDが高く、氷、睡眠、光、強回復を覚える。

### 特技

- 灯癒: HP回復。
- 星灯癒: 強いHP回復。
- 雷灯: 雷属性攻撃。
- 火打ち: 火属性攻撃。
- 氷針: 氷属性攻撃。
- 眠り霞: 睡眠付与。
- 星光: 光属性攻撃。
- 毒の裂き目: 敵専用。毒を伴う攻撃。
- 眠りの鏡面: 敵専用。睡眠付与。
- 黒曜の光: 敵専用。光属性攻撃。

- 潮読みの杖: ミオ初期武器。

- 星縁の刃: エルナ用光属性武器。
- 旅布の服: 初期防具。
- 硬革の上着: エルナ/カイ用防具。

- 露影: 序盤雑魚。火と光に弱い。

- 洞の薄影: 洞窟雑魚。睡眠攻撃。光と雷に弱い。
- 鉱脈の番片: 洞窟雑魚。硬い。雷に弱い。
- 潮鍵の番人: 中ボス。青の灯晶を守る。雷と光に弱い。
- 塔の仮面: 塔雑魚。睡眠攻撃。氷と光に弱い。
- 黒曜兵: 塔雑魚。高ATK。火と雷に弱い。
- 星喰いの鏡: ラスボス。属性結界と大技チャージを持つ。

## クリアまでの道筋

1. 灯丘の里で長老に話す。カイが加入する。
2. フィールドへ出て雫の洞窟へ向かう。
3. 洞窟の宝箱から古い鍵を取る。
4. 押し岩をスイッチに乗せ、鉄格子を開く。
5. 古い鍵で奥扉を開け、潮鍵の番人を倒す。青の灯晶を入手する。
6. 港へ行き、港長の選択を行う。灯台へ分けると潮灯の外套、温存すると280G。
7. ミオに話して仲間にする。
8. 必要なら里の薬師に星露草を渡し、サイドクエスト報酬を得る。
9. 港や里の店で回復アイテムと装備を整える。
10. 黒曜塔へ進む。星縁の刃と回復宝箱を回収する。
11. 星喰いの鏡と戦う。表示された弱点に合わせて火打ち、氷針、星光を使い、大技を止めながら倒す。
12. 撃破後、港長の選択とサイドクエスト結果が反映されたエンディングを見る。
